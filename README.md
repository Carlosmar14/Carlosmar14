## 🔥 Proyecto Destacado

### 🧾 POS SYSTEM

Sistema de punto de venta desarrollado con tecnologías modernas para la gestión de negocios.

### 🚀 Características

* 🛒 Gestión de ventas
* 📦 Control de inventario
* 👥 Gestión de usuarios
* 📊 Reportes básicos
* 💾 Persistencia con base de datos

### ⚙️ Tecnologías usadas

* React
* Node.js
* SQL
* Tailwind CSS

```bash

npm install
npm run dev

# Instalar backend
cd ../server
npm install
npm start
```
